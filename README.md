# YouTube Ad Skip 🟢

**Automatic YouTube ad skipper for Chrome.**

YouTube Ad Skip watches the page in the background and instantly skips, mutes, and fast-forwards through ads the moment they appear — no clicking required.

---

## 📥 Installation

This extension isn't on the Chrome Web Store, so it's installed manually in 3 quick steps:

### Step 1 — Download the ZIP
Click the green **Code** button at the top of this repo → **Download ZIP**, then unzip the folder anywhere on your computer.

### Step 2 — Open Chrome's Extensions page
Copy past this in url secction
In Chrome, go to:
```
chrome://extensions  
```
Turn on **Developer mode** using the toggle in the top-right corner of the page.

### Step 3 — Load the extension
Click **Load unpacked**, then select the unzipped `YouTube-Ad-Skip` folder.

That's it — the extension icon will appear in your toolbar. Pin it for easy access, and use the on/off switch in the popup whenever you want to pause it.

---

## ⚙️ Usage

Click the YouTube Ad Skip icon in your toolbar to open the popup. The switch is **ON by default** — flip it off any time to disable the ad-skipping behavior without removing the extension.

The popup also shows a small dashboard:
- **Skipped this tab** — resets each time you open a fresh YouTube tab
- **Total ads skipped** — an all-time running total that persists across browser restarts
